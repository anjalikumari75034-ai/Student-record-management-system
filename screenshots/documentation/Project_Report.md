# Student Record Management System

## Developed By
Anjali Kumari

## Duration
2 Weeks

## Language

## Objective
The objective of this project is to manage student records efficiently using C programming language and file handling concepts.

## Features
- Add Student Record
- Display Student Records
- Search Student
- Update Student Record
- Delete Student Record
- Find Topper
- Automatic Grade Calculation
- File Handling

## Student Information Stored
- Roll Number
- Name
- Course
- Marks
- Grade

## Technologies Used
- C Programming
- VS Code
- GitHub

## Algorithm
1. Start Program
2. Display Menu
3. Accept User Choice
4. Perform Selected Operation
5. Save Records
6. Exit Program
